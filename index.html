ℹ️ Instrucciones

  1. Haz clic en "Iniciar Seguimiento" para comenzar
  2. Comparte tu ubicación cuando el navegador lo solicite
  3. Muévate para ver el seguimiento en tiempo real
  4. Haz clic en "Detener Seguimiento" para finalizar

Nota: Para mejores resultados, usa un dispositivo con GPS.

⚙️ Configuración de Mapbox

Para usar esta aplicación, necesitas:

  1. Crear una cuenta en Mapbox
  2. Obtener un token de acceso público
  3. Reemplazar "TU_TOKEN_MAPBOX_AQUI" con tu token real
